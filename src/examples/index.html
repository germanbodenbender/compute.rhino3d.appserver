01 // ASSINGMENTS

COMPILATION OF EARLY EXCERCISES THAT SHOWS THE CONCEPTS OF HTML, JAVASCRIPT
AND CSS IMLEMENTED ON GITHUB PAGES AS WELL AS THE USE OF THREE.JS LIBRARIES



+A1-01
HTML PAGE
First seminar assignment where we learn how to create github pages and
push HTML and Javascript code to create our very first website.
+A2-01>
RHINO MODEL
Second seminar assignment where we learn to use the 3DM model loader from
the Three JS library.
+A2-02
VIEW COMMANDS
Exploration on different ways to navigate 3D models. Example from
the Three JS website.
+A3-01
RHINO COMPUTE
 Third assignment of the seminar where we explore the basics of Rhino.Compute
and how to bring our own GH definitions to the cloud.
+A3-02
RHINO COMPUTE
 Further exploration of the Rhino.Compute server using GH definitions.


02 // FINAL PROJECT

+FA 01
LUNAR PICNIC
This example allows you to set up your Lunar tent using Kangaroo and Ladybug components.
+FA 02
LUNAR SPACE RESIDENCY
Create your lunar community using modular 3D printed components
+FA 03
PARAMETRIC TOWER
Create your own version of the famous twisted tower and analyze it using radiation components.
